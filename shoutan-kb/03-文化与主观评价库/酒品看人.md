# 酒品看人 · 酒桌上的老手评价体系

> 本文件记录**资深人士在酒桌上"看人"的潜规则**。
> **重要提示**：本文件**不鼓励过量饮酒**。所有评价体系都应框定在**健康、自愿、尊重对方身体状况**的边界内。现代成熟圈层里，"能喝"早已不是核心评价维度，**"会喝、会拒、会照顾人"才是**。
> `[来源：坊间共识·酒品看人]`

---

## 一、酒桌的四个观察维度

老手在酒桌上**不是看你喝多少**，而是看这四件事：

| 维度 | 看什么 | 老手评价 |
|---|---|---|
| **座次处理** | 进门是否等指引、是否坐对位 | 懂不懂规矩 |
| **敬酒顺序** | 是否按位阶、是否跳人 | 识不识大体 |
| **拒酒方式** | 是硬刚、是扭捏、还是高情商化解 | 成不成熟 |
| **醉后表现** | 失态与否、是否管得住嘴 | 可不可托大事 |

> **现代成熟圈层里，"酒量"早已不是评价核心。真正被看的是：懂座次、识顺序、会拒酒、醉后有体面。**

## 三、几种典型"酒桌人格"（老手分类）

| 类型 | 特征 | 老手评价 |
|---|---|---|
| **照顾型** | 全程倒酒、布菜、接话、安排车辆 | "可深交，会来事" |
| **豪爽型** | 大口干、劝酒猛、讲义气 | "够朋友，但需注意分寸" |
| **节制型** | 以茶代酒、态度到位、从不失态 | "有定力，理性" |
| **躲酒型** | 扭捏、躲闪、找借口 | "不爽快，难交心"（过度推论）|
| **强劝型** | 硬灌、激将、道德绑架 | "霸道，不尊重人" |
| **失态型** | 醉后乱说话、哭闹、撒泼 | "不可托大事" |

## 四、酒桌的"红线行为"（一票否决）

| 行为 | 后果 |
|---|---|
| **醉后辱骂他人** | 关系断裂，难以挽回 |
| **醉后泄露机密/八卦** | 信任归零，可能涉及职业风险 |
| **醉后对异性失礼** | 严重，可能涉及法律 |
| **强行灌醉他人致健康损害** | 法律责任 + 关系断裂 |
| **借酒装疯侮辱上位者** | 社交自杀 |
| **酒驾** | 法律红线，绝不可碰 |

`[来源：坊间共识·酒桌红线]`

> **健康提醒**：上述"红线"中的若干项已不仅是社交失格，而是**法律与伦理的硬底线**。任何社交场合都不应以健康、安全、法律为代价。

## 五、给"手谈"的使用提醒

1. **酒桌不是"比酒量"，是"比懂事"**——你全程以茶代酒但仪式到位，比强行硬喝更得人心。
2. **拒酒要趁早、要一致**——今晚不喝，就别破例；破例一次，下次难拒。
3. **醉后管住嘴**——这是酒桌最重要的 KPI。**宁可早散，不可失言**。
4. **散场的收尾比开场的敬酒更见功夫**——安排车、送客、确认 everyone 安全到家，是"有始有终"的标志。
