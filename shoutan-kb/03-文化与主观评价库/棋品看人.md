## 一、围棋：看"格局、远见、节奏感"

围棋是中国精英阶层最认可的智力符号。**棋风几乎被视为性格的投影**。

| 棋风 / 行为 | 老手的潜台词评价 | 推论属性 |
|---|---|---|
| 布局沉稳，重视实地 | "务实，有战略定力" | 【坊间共识】 |
| 喜欢大模样、宇宙流 | "有格局，但可能好高骛远" | 【过度推论】 |
| 中盘计算精准 | "思维缜密，能扛大事" | 【坊间共识】 |
| 攻击性强，爱挑战斗 | "有胆识，但需看是否莽撞" | 【过度推论】 |
| 防守稳健，厚味派 | "稳重，可托底" | 【坊间共识】 |
| 收官细腻，不丢一目 | "细节控，做事讲究" | 【坊间共识】 |
| 悔棋 | "不诚信，不守规矩" | 【坊间共识·硬禁忌】 |
| 死撑不认输 | "不识时务，固执" | 【过度推论】 |
| 早认输过多 | "缺韧性，易放弃" | 【过度推论】 |
| 让子时坦然受让、认真下 | "心态好，可教" | 【坊间共识】 |
| 让子后赢了上位者还炫耀 | "不识趣，情商低" | 【坊间共识】 |

### 经典对照：棋风与性格

| 棋风 | 对应性格画像（老手眼中的）|
|---|---|
| 实地派（先捞地再说）| 务实、重眼前、踏实 |
| 厚势派（先做厚再发力的）| 隐忍、有后劲、城府 |
| 战斗派（爱中盘对决）| 果断、有野心、可能冲动 |
| 收官派（细节决胜）| 精细、严谨、可能纠结 |

### 一句话总结
> **围棋是中国精英阶层最认可的智力符号。棋风即性格的投影——布局看眼界，中盘看算度，收官看耐心。**

## 二、象棋：看"算度、耐心、识时务"

象棋是民间最普及的智力游戏，**公园大爷、机关干部、商界老总通吃**。

| 棋风 / 行为 | 老手的潜台词评价 | 推论属性 |
|---|---|---|
| 开局出子迅速 | "执行力强，不拖沓" | 【坊间共识】 |
| 中盘攻防转换流畅 | "思路清晰，应变快" | 【坊间共识】 |
| 残局算度深（3 步以上）| "思维缜密，有远见" | 【坊间共识】 |
| 敢于计算后弃子入局 | "有战略胆识" | 【坊间共识】 |
| 乱弃子，无后续 | "莽撞，不计后果" | 【过度推论】 |
| 走一步看一步 | "短视，缺规划" | 【过度推论】 |
| 残局死撑不认输 | "不识时务，固执" | 【过度推论】 |
| 残局早认输 | "缺韧性" | 【过度推论】 |
| 长将、长捉（违规）| "爱走捷径，不守规矩" | 【坊间共识】 |
| 悔棋 | "不诚信" | 【坊间共识·硬禁忌】 |
| 让子/让先后，赢了不张扬 | "识趣，懂规矩" | 【坊间共识】 |

### 经典对照：象棋开局与性格

| 开局选择 | 老手的性格联想 |
|---|---|
| 中炮（进攻型）| 果断、有野心、外向 |
| 屏风马（稳健型）| 稳重、内敛、有定力 |
| 反宫马（弹性型）| 灵活、善应变 |
| 仙人指路（试探型）| 谨慎、善观察 |

### 一句话总结
> **象棋看的是算度与识时务。能算三步以上者多缜密，死撑不认者多固执，让子后识趣者多得人心。**

## 四、给"手谈"的使用提醒

1. **棋桌是"上位者下指导棋"的经典场景**——受让子、被点拨，是下位者展示"可教、有悟性"的窗口。
2. **棋风评价有其局限**——棋盘上的风格不等于职场上的全部，但**它是老手认知中最直观的"智力与性格样本"**。
3. **你不必追求棋力高强**——你只需表现出"懂规矩、有思考、识分寸"，就已在老手滤镜里过关。
