# 桥牌 · 硬规则速记

> 来源标注：按 **Contract Bridge（合约桥牌）国际通行规则**整理，使用精确叫牌体系（如 SAYC、Precision 等）。
> `[来源：桥牌·硬规则·通行版本]`

---

## 一、一句话判词
桥牌是**信息完全公开但只对搭档公开一半**的合作博弈。叫牌是"用代号交换情报"，打牌是"兑现你叫出去的合约"。**学圈、智识圈最爱的游戏**，因为它几乎是数学+沟通+合作的合体。

---

## 二、基本配置

| 项 | 规则 |
|---|---|
| 人数 | 4 人，南北 vs 东西，对面为搭档 |
| 用牌 | 一副 52 张，无大小王 |
| 抓牌 | 每人 13 张 |
| 阶段 | 叫牌（Auction）→ 打牌（Play）→ 计分 |
| 花色顺位（从高到低）| 黑桃 ♠ > 红心 ♥ > 方块 ♦ > 梅花 ♣ > NT（无将）|

> 注：NT 实际为最高级别合约，但在花色比较中"无将"不参与花色顺序。

## 三、叫牌（Auction）—— 桥牌的灵魂

### 1. 大牌点（HCP，High Card Points）
- A = 4 点，K = 3 点，Q = 2 点，J = 1 点
- 全副共 40 点。开叫一般需要 **12+ HCP**。
- **记住一句话**：13 点以下通常不主动开叫，除非有长套或分配点。

### 2. 叫品的含义（以 SAYC 精确体系为例）
| 叫品 | 一般含义 |
|---|---|
| 1NT 开叫 | 平均牌型，15–17 HCP |
| 1♣/1♦ 开叫 | 12+ HCP，所叫花色 ≥3 张 |
| 1♥/1♠ 开叫 | 12+ HCP，所叫花色 ≥5 张 |
| 2NT 开叫 | 平均牌型，20–21 HCP |
| 应叫 1NT | 6–9 HCP，对开叫无支持 |
| 加叫搭档花色到 2 阶 | 6–9 HCP + 3 张支持 |
| 跳加叫到 3 阶 | 10–12 HCP + 支持，邀请 |
| 双跳/扣叫 | 进局试探或控制显示 |

`[来源：桥牌·硬规则·通行版本-SAYC]`

### 3. 合约的构成
- **阶数（1–7）+ 花色/无将**：如 3NT、4♠。
- 阶数 × 6 + 6 = 该合约需要赢的**墩数**。例：3NT = 3+6 = **9 墩**。
- 7NT 满贯需赢全部 13 墩。

## 四、打牌（Play）

### 1. 定约人与明手
- 叫牌胜出方为**定约方**，先叫出该花色/无将的人为**定约人**。
- 定约人对面的搭档摊开全部 13 张牌为**明手（Dummy）**，由定约人代打。
- 防守方两人牌面朝下，凭记忆与推理打。

### 2. 跟牌规则
- **必须跟出首攻花色**（有则必跟）。
- 无该花色才能垫牌或将吃（无将合约无将吃）。
- 每墩 4 张，最大者赢，赢者首攻下一墩。

### 3. 关键术语
- **首攻（Opening Lead）**：定约人左手防守者打出的第一张牌，往往是合约成败的关键。
- **飞牌（Finesse）**：试图用次高牌赢墩的高级技巧，如手握 AQ，希望 K 在对手某一侧。
- **将吃（Ruff）**：无该花色时打出将牌赢墩。

## 五、计分（极简版）

| 情况 | 得分 |
|---|---|
| 完成花色/NT 合约 | 按阶数与花色算分（黑桃红心每墩 30，梅花方块每墩 20，NT 首墩 40，余 30） |
| 部分定约 | <100 分 |
| 成局（Game）| ≥100 分，有局况奖励 |
| 小满贯（6 阶）| 重奖 |
| 大满贯（7 阶）| 顶级重奖 |
| 宕（Down）| 未完成合约，每墩宕罚 50（无局）/ 100（有局），有放大规则 |

## 六、新手最容易踩的规则雷区

1. **叫牌暴露点力**：把搭档叫品当"明信片"读，泄露自己手牌强度（桥牌规则禁止"非法传递信息"，违规判罚）。
2. **首攻选错**：首攻一张错牌，整副牌的攻防逻辑可能全乱。
3. **飞牌概率算错**：手握 AQ，希望 K 在某一侧——概率 50%，但需要看清已出牌信息调整。
4. **不跟牌**：明明有该花色却垫别的——技术上违规，社交上极度尴尬。
5. **明手"伸手"**：明手只能由定约人代打，明手说话/指挥是大忌。
6. **不看局况（Vulnerability）**：有局宕墩罚得狠，策略完全不同。

## 七、定位判词模板

- **下位陪打**：桥牌对搭档默契要求极高。**做搭档时，叫牌要"安全"**——不要把上位者拖入他/她不想进的合约；做对手时，**不要在叫牌里给上位者制造难堪**。
- **平位切磋**：桥牌是看智商的窗口。叫牌是否清晰、防守是否严密、面对飞牌的概率判断是否稳健——**3 副牌就能看出对方是不是"系统思考型"选手**。
- **上位考察**：观察搭档沟通是否干净（无非法信息传递）、面对宕局是否冷静、是否敢于承担满贯风险。**桥牌是识人最准的智力游戏之一**。
