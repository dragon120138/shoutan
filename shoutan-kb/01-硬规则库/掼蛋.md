# 掼蛋 · 硬规则速记

> 来源标注：本文件按**公开通行的国家/地方掼蛋竞赛规则**整理，强调"通行"而非某一地方版本。地方细则（如淮安、镇江、南京、江苏省级规则）存在出入，**临场以主家或当地现行版本为准**。
> `[来源：掼蛋·硬规则·通行版本]`

---

## 一、一句话判词
掼蛋是**四人、两副牌、两两结对**的升级类扑克。胜负不是"我赢"，而是"我们这一对赢了"。社交属性极强，**对门是命运共同体**。

---

## 二、基本配置

| 项 | 通行规则 |
|---|---|
| 人数 | 4 人，对面为对门（搭档） |
| 用牌 | 两副扑克共 108 张 |
| 抓牌 | 每人 27 张 |
| 目标 | 率先**打完手中所有牌**的一方为胜；打到对方没人能压为"上游"机制 |
| 升级 | 胜方按上游/二游/三游/下游（俗称"头锅～末锅"）的成绩升级，从 2 打到 A，过 A 即为终局 |

## 三、核心牌型（必须背熟）

| 牌型 | 构成 | 备注 |
|---|---|---|
| 单张 | 1 张 | |
| 对子 | 同点数 2 张 | |
| 三张 | 同点数 3 张 | |
| 三带二 | 3 + 同点数对 | "三带一对" |
| 三连对 | 三组相连的对子，如 334455 | |
| 三同连张（钢板）| 两组相连的三张，如 333444 | |
| 顺子 | 5 张连续单张（A 可作大小头，但通常不可作顺子两头） | |
| 同花顺 | 同花色 5 张连续 | 比一般顺子大 |
| 炸弹 | 4 张及以上同点 | 压一切非炸弹牌型 |
| 同花炸（"火箭/同花顺炸弹"）| 同花色 4 张及以上 | 部分版本中为最高级别炸弹之一 |
| 王**（"对王"）| 大王 + 小王 | 通行规则中为**最高炸弹** |

`[来源：掼蛋·硬规则·通行版本]`

> ⚠️ **地方差异提示**：王**是否最大、同花炸与普通炸弹孰高、顺子能否绕过 A、能否"三带两单"等，各版本规则不同。**开打前一定要先和主家确认一句"今天按哪版规则"**——这是社交场合显出"懂行"的细节。

## 四、关键机制

### 1. 进贡与还贡
每局开始时，上一局的下游（输方）须向对方进贡**最大的牌**（一般不含王），对方还一张任意牌（通常还小牌）。
- 上游、下游、双下等不同结果，进贡张数与方式不同。
- 进贡是掼蛋**社交润滑剂**：会进贡、会还贡，是"懂规矩"的第一标志。

### 2. "对门不打" / 搭档机制
- 对门之间不互相压牌，对门打的牌型你只需考虑是否要顶住对方两人。
- 对门出牌，你不抢；对门压不住对方，你再上。

### 3. 牌权与顺牌
- 一手牌若四家都不压，则**最后出牌者获得下一手出牌权**（俗称"牌权")。
- 抢到牌权意味着你能主导出牌方向，往往比一手大牌更值钱。

## 五、新手最容易踩的规则雷区（高频翻车点）

1. **顺子认错**：把 `Q K A 2 3` 当顺子出——**2 不能连**，多数版本顺子止于 A。
2. **同花顺 / 普通顺子混压**：普通顺子不能压同花顺；同花顺之间比最大单张点数。
3. **炸弹大小搞反**：王** > 5 张炸 > 4 张炸（具体顺位以当地版本为准，开打前问一句）。
4. **三带二带两张单**：很多版本**只允许带一对、不能带两单**，新手常错。
5. **进贡忘进 / 进错牌**：进贡必须进"最大牌"，进了第二大会被罚（部分版本）。
6. **对门牌抢压**：对门领出的牌你伸手去压——这是社交场合的"不懂规矩"大忌。
7. **下游回贡不会回**：还贡应还对方**接得住、用得上**的小牌，不是随手扔。

## 六、定位判词模板（供"手谈"快速调用）

- **下位陪打**：你的 KPI 不是上游，而是**把对门领导伺候成 MVP**。让他牌权顺、让他压牌爽、让你自己甘当最佳配角。
- **平位切磋**：识别对方是不是"炸弹堆里苟"型还是"牌权主导"型，决定你按哪种节奏打。
- **上位考察**：观察搭档与对手的反应——谁会进贡、谁压牌鲁莽、谁对门牌伸手，**这是看人最快的窗口**。
