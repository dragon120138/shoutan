# 象棋 · 硬规则速记

> 来源标注：按**中国象棋通行规则**整理。
> `[来源：象棋·硬规则·通行版本]`

---

## 一、一句话判词
**两人、楚河汉界、16 子对 16 子，以将死对方"将/帅"为胜。** 象棋是中国民间最普及的智力游戏，**公园大爷、机关干部、商界老总通吃**，是"老一辈"最熟悉的社交媒介。

---

## 二、基本配置

| 项 | 规则 |
|---|---|
| 人数 | 2 人（红先黑后） |
| 棋盘 | 9×10，中间"楚河汉界" |
| 棋子 | 每方 16 子：帅/将 ×1、仕/士 ×2、相/象 ×2、马 ×2、车 ×2、炮 ×2、兵/卒 ×5 |
| 目标 | **将死**对方的将/帅（被将军且无法解将） |

`[来源：象棋·硬规则·通行版本]`

## 三、各子走法（必须背熟）

| 棋子 | 走法 | 备注 |
|---|---|---|
| **帅/将** | 九宫内，每步直走或横走一格 | 不可斜走，不可出九宫 |
| **仕/士** | 九宫内，每步斜走一格 | 不可直走 |
| **相/象** | 走"田"字（斜两格），不能过河 | 中间有子则"塞象眼"不能走 |
| **马** | 走"日"字（直一格再斜一格）| 直行方向有子则"蹩马腿"不能走 |
| **车** | 直线任意距离 | 攻击力最强 |
| **炮** | 直线任意距离，**吃子须隔一子**（"炮架"）| 唯一需要"架"才能吃子 |
| **兵/卒** | 过河前只能直走一格；过河后可直走或横走一格 | 永远不能后退 |

`[来源：象棋·硬规则·通行版本]`

## 四、关键机制

### 1. 将军与将死
- **将军**：下一步可吃将/帅。
- **应将**：被将军时**必须**应将（躲、挡、吃掉将军子），不能"不理"。
- **将死**：无法应将，败。
- **困毙**：无棋可走（虽未被将军但所有合法走法都会自入绝境），判负。

### 2. 长将禁手
**不可反复将军**（同一着法连续将军）——长将判负。

### 3. 长捉禁手
- 重复"捉子"（威胁吃子）在某些规则中受限，**亚洲规则禁止长捉**。
- 友谊局一般宽松，但正式比赛严格。

## 五、胜负与和棋

| 情况 | 结果 |
|---|---|
| 一方被将死 | 另一方胜 |
| 一方主动认输 | 另一方胜 |
| 双方均无力将死 | 和棋 |
| 一方提出和棋，另一方同意 | 和棋 |
| 50 回合无吃子（部分规则）| 和棋 |
| 长将、长捉 | 判负（或和，视规则）|

## 六、新手最容易踩的规则雷区

1. **马蹩腿**：马走"日"时若直行方向有子（蹩马腿），不能走。新手常忽略。
2. **象塞眼**：象走"田"时田字中心有子（塞象眼），不能走。
3. **炮没架**：炮吃子必须隔一子，平移（不吃子）则无需架。
4. **兵后退**：兵/卒**永远不能后退**，新手常误走。
5. **帅照面**：双方将/帅不能在同一直线上无遮挡相对（"老王见面"），新手易忽略。
6. **长将不自知**：反复将军以为是在进攻，实际违规。
7. **悔棋**：与围棋同，社交场合硬禁忌。

`[来源：坊间共识·新手高频翻车点]`

## 七、定位判词模板

- **下位陪打**：象棋是"老一辈"最熟悉的话题。**让先**（你不先行）或**让子**（让车马炮）是上位者给你的体面；你的反应应是"欣然接受、认真下、输了显出学习态度"。
- **平位切磋**：象棋是看"算度"和"耐心"的窗口——**能算三步以上**的人，做事通常也周密。
- **上位考察**：象棋最能看出一个人**是否急躁、是否识大局**。开局求稳、中盘求势、残局求精的人，往往是成熟型；开局猛冲、不重后势者，多半性格冲动。
